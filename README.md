# Radiance Labs

Site for **Radiance Labs** — a non-commercial AI research laboratory studying
Semantic Gaussian Splatting (SGS): foundational model research and
prompt-to-3D Gaussian-splat scene generation.

Live: https://feamando.github.io/radiance-labs/

## Structure

- `index.html` — single-file static site (Inter + Source Serif 4, WebGL splat hero).
- `papers/` — paper PDFs linked from the site.
- `.nojekyll` — serve files as-is (no Jekyll processing).

## Hosting

GitHub Pages, served from the `main` branch root. Static only — no backend.

Built with love in Berlin. Research code: https://github.com/feamando/sgs
